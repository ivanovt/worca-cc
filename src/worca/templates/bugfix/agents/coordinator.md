## Override: Task Decomposition

Create 1-3 focused fix tasks only. Do not decompose broadly — this is a bug fix, not a feature. Each task must directly address the root cause identified by the planner.

- Prefer a single task if the fix is self-contained.
- Split into multiple tasks only if the fix spans clearly separate modules.
- Do not add tasks for refactoring, cleanup, or unrelated improvements.
