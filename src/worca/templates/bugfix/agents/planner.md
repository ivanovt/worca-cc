## Override: Objective

Investigate this bug, identify root cause, and scope the minimal fix needed. Do not propose new features or broad refactors — stay tightly focused on the reported problem.

## Override: Approach

1. Reproduce the bug by tracing the code path that causes it.
2. Identify the root cause (not just the symptom).
3. Determine the minimal change required to fix it without introducing regressions.
4. Document findings clearly so the coordinator can create focused implementation tasks.
